Slider Images Directory
========================

This directory should contain the following WebP images for the hero section slider:

Required files:
- slide1.webp
- slide2.webp  
- slide3.webp

Image specifications:
- Format: WebP for optimal performance
- Recommended dimensions: 800x600px or higher (16:9 or 4:3 aspect ratio)
- Content: Professional business/accounting related images
- File size: Optimized for web (typically under 200KB each)

Usage:
These images are used in the HeroSection component's auto-playing slider.
The first image (slide1.webp) loads with priority for better performance.
Images automatically cycle every 4 seconds with smooth fade transitions.
